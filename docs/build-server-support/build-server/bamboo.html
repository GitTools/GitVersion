Bamboo

If you use Bamboo then you will have to use GitVersion from the command line, as there is no actively supported app. You can use the "Inject Bamboo Variables" task to read the GitVersion output back into Bamboo. Below are two examples using the .NET Core GitVersion global tool.

Example

The first script can be run on a Linux build host, the second script can be run on a Windows build host using Powershell. The build only needs one of the two.

Task: Script

Script body

~/.dotnet/tools/dotnet-gitversion > gitversion.txt
sed -i '1d;$ d;s/  //;s/"//g;s/,//;s/:/=/' gitversion.txt

Task: Script

Script body

(dotnet gitversion | ConvertFrom-Json).PSObject.Properties | ForEach-Object { Write-Output "$($_.Name)=$($_.Value)" } | Out-File -Encoding UTF8 -FilePath gitversion.txt

Task: Inject Bamboo variables Configuration

Required Properties

  • Path to properties file: gitversion.txt
  • Namespace: GitVersion
  • Scope of the Variables: Result
